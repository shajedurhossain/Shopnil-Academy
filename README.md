## About

Shopnil Academy is a free online learning platform built on a simple principle: education shouldn't have a paywall. Every course — past, present, and future — is completely free. No subscriptions, no locked lessons, no credit card required.

Rather than launching with hundreds of mediocre courses, Shopnil Academy is built carefully, one well-crafted course at a time. Each course is structured around real projects and designed to genuinely teach.

## Running Locally

No build step required — it's plain HTML/CSS/JS.

```bash
git clone https://github.com/your-username/shopnilacademy.git
cd shopnilacademy

# Open in browser directly, or use a local server:
npx serve .
# or
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
